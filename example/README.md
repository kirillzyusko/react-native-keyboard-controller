# Example

## Run the app

<img src="../gifs/example-screenshot.png" align="right" width="35%" style="margin-right: 40px;">

1. Install the repository dependencies (from the project root):

```bash
yarn
```

2. Install the example app dependencies:

```bash
cd example
yarn
```

3. Start the Metro server:

```bash
yarn start
```

4. In a separate terminal, build and install the app:

```bash
yarn android
# or
yarn ios
```
