# ChatKit

`ChatKit` is a specialized component designed to simplify the process of building chat application layouts that handle keyboard interactions seamlessly. It addresses common challenges faced when developing chat apps, such as managing keyboard appearance, interactive dismissal, and ensuring smooth animations.

### `ScrollViewComponent`

Custom component that will be used as a `ScrollView`. Default is `ScrollView`.

:::tip When to use it?
If you want to use `ScrollView` from `react-native-gesture-handler` you can pass it as a `ScrollViewComponent` prop.

```tsx
import { ScrollView } from "react-native-gesture-handler";

<KeyboardAwareScrollView ScrollViewComponent={ScrollView} />;
```

:::
